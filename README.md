# label-generator

Browser-based packaging label generator for LifeGoods Group B.V.

The application supports two main workflows:

- generating labels from manual single-item input;
- generating labels in bulk from uploaded data files.

## What this project contains

This project is a static front-end application built with HTML, CSS and JavaScript.
It generates printable label previews and PDF output in the browser.

Main files:

- `index.html` — application entry point
- `styles.css` — UI styling
- `app.js` — application logic
- `NOTICE` — third-party notices and licenses
- `LICENSE` — proprietary software terms
- `LICENSES/...` — customer-specific commercial license

## Usage

Open `index.html` in a modern desktop browser.

The application provides:

- a **Single-PDF** workflow for individual label generation;
- a **Bulk-Upload** workflow for batch processing via spreadsheet upload;
- a **Help** page with end-user instructions and operational notes.
